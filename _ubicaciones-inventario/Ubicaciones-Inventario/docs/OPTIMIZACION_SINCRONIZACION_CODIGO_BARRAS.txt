⚡ OPTIMIZACIÓN: SINCRONIZACIÓN POR UBICACIÓN SIN TIMEOUT
═══════════════════════════════════════════════════════════════════

❌ PROBLEMA IDENTIFICADO:
   La sincronización por ubicación se caía por TIMEOUT porque:
   
   - Hacía 1 llamada HTTP a /api/Items/{item_id} por CADA producto sin código de barras
   - Si ADESA tiene 6,451 productos sin código de barras = 6,451 llamadas HTTP individuales
   - Cada llamada toma ~200-500ms = 6,451 × 400ms = ~43 minutos de tiempo total
   - cPanel tiene timeout de ~30 segundos → PROCESO MUERTO

✅ SOLUCIÓN OPTIMIZADA:

1. NO hacer llamadas individuales a /api/Items durante sync por ubicación
   ✓ Solo usar código de barras si viene en la respuesta de /api/Stock
   ✓ Si NO viene, dejar el campo vacío (se obtiene después)

2. Separar responsabilidades:
   ✓ Sincronización por ubicación → RÁPIDA, solo stock
   ✓ Sincronización masiva de productos → Obtiene códigos de barras de forma eficiente

3. Código de barras se obtiene DESPUÉS:
   ✓ Con sincronización masiva de productos (paginación eficiente: 50 productos por llamada)
   ✓ Ejemplo: 6,451 productos = ~130 llamadas (6,451 ÷ 50) = mucho más rápido

📊 COMPARACIÓN DE RENDIMIENTO

   - 6,451 llamadas a /api/Items/{item_id}
   - 6,451 × 400ms = ~43 minutos
   - TIMEOUT en cPanel (30 segundos) ❌

✅ ENFOQUE OPTIMIZADO (sin llamadas individuales):

   - Códigos de barras se obtienen después con sync masiva ✅

   1. Obtener stock de /api/Stock?LocationID=...&skip=0&take=50
   2. Para cada item:

      ✓ Intentar extraer código de barras de la respuesta de Stock

      ✓ Guardar stock en StockProductoADM
   3. Commit periódico cada 100 items
   
   RESULTADO: Rápido, sin llamadas individuales ✅

SINCRONIZACIÓN MASIVA DE PRODUCTOS (PARA CÓDIGOS DE BARRAS):
   1. Obtener productos de /api/Items?skip=0&take=50 (paginación eficiente)
   2. Para cada lote de 50 productos:
      ✓ Extraer código de barras de la respuesta (ya viene incluido)
      ✓ Actualizar productos en BD con código de barras
   3. Commit periódico
   
   RESULTADO: Eficiente, 50 productos por llamada ✅
