📋 ENDPOINTS DISPONIBLES
===========================================================

POST /api/transferencias/buscar
→ Busca una transferencia por DocID en ADM Cloud
→ Resuelve nombres de ubicaciones desde cache local
→ Devuelve Origen → Destino y productos

GET /transferencias
→ Página HTML del módulo de Transferencias

===========================================================
🔗 INTEGRACIÓN CON ADM CLOUD
===========================================================

Endpoints ADM Cloud utilizados:

- GET /api/LocationTransfers (listar transferencias)
- GET /api/LocationTransfers/{GUID} (detalle de transferencia)

Estructura:
- DocType: "INV_TRA"
- LocationID → ReceptionLocationID (Origen → Destino)
- Items[] con productos transferidos

===========================================================
📌 MAPEO DE UBICACIONES
===========================================================

El sistema usa SyncLocationStatus para mapear:
- LocationID (GUID) → LocationName (ej: "ADESA")
- ReceptionLocationID (GUID) → LocationName (ej: "Mirador Sur")

Si no encuentra en cache, muestra parcial del GUID como fallback.
