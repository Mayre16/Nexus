===========================================================
📋 ENDPOINTS NUEVOS DISPONIBLES
===========================================================

POST /api/recepciones/buscar
→ Busca una recepción por DocID en ADM Cloud

POST /api/recepciones/registrar
→ Registra recepción asignando ubicaciones físicas
→ Crea movimientos tipo RECEIPT
→ Incrementa StockUbicacion

GET /recepciones
→ Página HTML del módulo de Recepción

===========================================================
🔗 INTEGRACIÓN CON ADM CLOUD
===========================================================

Endpoints ADM Cloud utilizados:
- GET /api/Receptions (listar recepciones)
- GET /api/Receptions/{ID} (detalle de recepción)

Campos importantes:
- DocID: Número de recepción
- Items[]: Productos recibidos
  - ItemID, ItemSKU, Name, Quantity, PendingCompletedQuantity
