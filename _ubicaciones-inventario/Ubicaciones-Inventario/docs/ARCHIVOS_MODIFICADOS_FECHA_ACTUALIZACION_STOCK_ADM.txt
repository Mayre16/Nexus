ARCHIVOS MODIFICADOS PARA MOSTRAR FECHA DE ACTUALIZACIÓN EN STOCK ADM
======================================================================

Fecha: 2026-01-26
Descripción: Agregar fecha de última actualización en la sección "Ubicaciones ADM Cloud" de la página "Buscar Producto"

ARCHIVOS MODIFICADOS:
---------------------

1. routes/productos.py
   - Líneas 136-140: Agregado campo `updated_at` al diccionario de ubicaciones ADM

2. templates/productos.html
   - Líneas 581-588: Modificado para mostrar la fecha de actualización en cada ubicación ADM

CAMBIOS REALIZADOS:
-------------------

1. Backend (routes/productos.py):
   - Agregado `updated_at` al diccionario de ubicaciones ADM
   - Formato: ISO string o None si no existe

2. Frontend (templates/productos.html):
   - Agregada visualización de fecha de actualización debajo del nombre de cada ubicación ADM
   - Formato: "Actualizado: DD/MM/YYYY" (formato español)
   - Similar al formato usado en "Ubicaciones Físicas WMS"

COMPORTAMIENTO:
---------------

**ANTES:**
- Ubicaciones ADM Cloud mostraban solo: Nombre y Stock
- Ejemplo: "En mano (ADESA): 8.00"

**DESPUÉS:**
- Ubicaciones ADM Cloud muestran: Nombre, Fecha de actualización y Stock
- Ejemplo:
  ```
  En mano (ADESA)
  Actualizado: 27/1/2026
  8.00
  ```

**Consistencia:**
- Ahora ambas secciones (ADM Cloud y WMS Físicas) muestran la fecha de actualización
- Formato consistente en ambas secciones
