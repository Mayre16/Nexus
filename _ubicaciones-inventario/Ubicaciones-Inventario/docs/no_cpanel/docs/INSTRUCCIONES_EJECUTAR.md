# 🚀 Cómo Ejecutar la Aplicación

## Pasos para Ejecutar

### 1. Abre una Terminal/PowerShell

- Presiona `Win + X`
- Selecciona "Windows PowerShell" o "Terminal"
- Navega a la carpeta del proyecto:
  ```powershell
  cd C:\Proyectos\Ubicaciones-Inventario
  ```

### 2. Ejecuta la Aplicación

```powershell
python app.py
```

### 3. Deberías Ver Esto:

```
============================================================
Consultor ADM Cloud API
============================================================

Iniciando servidor...
Abre tu navegador en: http://localhost:5000

Presiona Ctrl+C para detener el servidor
============================================================
 * Running on http://0.0.0.0:5000
 * Debug mode: on
```

### 4. Abre tu Navegador

Abre cualquier navegador y ve a:
```
http://localhost:5000
```

### 5. Para Detener el Servidor

En la terminal donde está ejecutándose, presiona:
```
Ctrl + C
```

## ⚠️ Notas Importantes

- **Mantén la terminal abierta** mientras uses la aplicación
- Si cierras la terminal, el servidor se detendrá
- Si ves errores, compártelos para poder ayudar

## 🔧 Si hay Errores

Si al ejecutar `python app.py` ves algún error:

1. Verifica que Flask esté instalado:
   ```powershell
   pip install Flask requests
   ```

2. Verifica que estés en la carpeta correcta:
   ```powershell
   cd C:\Proyectos\Ubicaciones-Inventario
   ```

3. Comparte el mensaje de error completo
