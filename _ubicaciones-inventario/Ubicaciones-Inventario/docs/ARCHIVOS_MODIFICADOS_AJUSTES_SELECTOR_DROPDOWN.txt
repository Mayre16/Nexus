1. Subir archivo modificado:
   - templates/ajustes.html

   - Tocar el archivo tmp/restart.txt
   - O usar "Restart App" en el panel de Python App

3. Verificar:
   - El selector desplegable muestra ubicaciones ADM disponibles
   - Al seleccionar una ubicación, se agrega a la lista con campo de cantidad
   - Al seleccionar ADESA, aparece sección de ubicaciones físicas
   - La validación bloquea el botón si suma físicas ≠ cantidad ADESA
   - Se muestran advertencias detalladas sobre ubicaciones con stock no ajustadas

FUNCIONALIDADES IMPLEMENTADAS:
==============================

COMPORTAMIENTO DE VALIDACIÓN:
==============================

1. Si ADESA = 30 y suma de físicas ≠ 30:
   - Bloquea botón "Registrar Ajuste"
   - Muestra error con diferencia
   - Lista ubicaciones físicas con stock que no están siendo ajustadas

2. Si hay ubicaciones físicas con stock que no se están ajustando:
   - Bloquea botón "Registrar Ajuste"
   - Muestra lista de ubicaciones con stock no ajustadas
   - Ejemplo: "2P1D01N3 tiene 5"

3. Solo permite ajuste si:
   - Suma de ubicaciones físicas = cantidad ADESA
   - Todas las ubicaciones físicas con stock están siendo ajustadas

NOTAS IMPORTANTES:
==================

- El sistema NO permite discrepancias en ajustes
- Si hay ubicaciones físicas con stock que no se ajustan, el sistema bloquea el ajuste
- Se sugiere sincronizar con ADM para resolver discrepancias
- Las ubicaciones físicas dentro de ADESA se muestran en la información del producto
